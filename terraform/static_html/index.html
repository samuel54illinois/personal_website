Project 1: Global Superstore Sales Analysis

This project involves a comprehensive analysis of the Global Superstore dataset to derive business insights. Using Tableau, I created eight interactive visualizations and a consolidated dashboard to explore sales, profit trends, shipping costs, and customer behaviors. The project demonstrates skills in data cleaning (Python), exploratory analysis (Jupyter), and data visualization (Tableau Public). Key takeaways include identifying regional performance disparities, product category profitability, and customer segmentation for strategic business decisions. Project 2: Mall Customer Segmentation

Description: This project uses K-means clustering to segment mall customers based on their annual income and spending behavior. The analysis identifies four distinct customer groups, each with unique characteristics, to guide marketing strategies. New Highlights: Added actionable business recommendations for each customer segment based on insights from the clustering. Included visuals, such as the Elbow Method plot and customer segmentation scatterplot, for better data representation. Business Impact: Budget-friendly strategies for price-sensitive customers. Premium offerings and personalized experiences for high-value customers. Promotions and flexible payment options for high-spending customers.

Technologies: Python (Programming Language) · Machine Learning · Data Science
